Format danych GTFS

General Transit Feed Specification

  • agency.txt agencje, firmy obsługujące kursy
  • stops.txt lokalizacje przystanków
  • routes.txt trasy wraz terminem ważności
  • trips.txt wszystkie kursy
  • stop_times.txt tabela czasów przystankowych
  • calendar.txt czas obowiązywania rozkładu, brygady
  • niewymagane: calendar_dates.txt, fare_attributes.txt, fare_rules.txt, shapes.txt, frequencies.txt, transfers.txt, feed_info.txt

GTFS tips

preprocessing stop_times.txt === ~32 MB

HH:mm (kurs od godz. 23:55 do 25:01)

validatory

transitfeeds.com

GTFS MPK Wrocław 03/03/2017

  • control_stops.txt: Unrecognised file in feed: control_stops.txt
  • route_types.txt: Unrecognised file in feed: route_types.txt
  • routes.txt: Unrecognised column in routes.txt: route_type2_id
  • routes.txt: Unrecognised column in routes.txt: valid_from
  • routes.txt: Unrecognised column in routes.txt: valid_until
  • trips.txt: Unrecognised column in trips.txt: brigade_id
  • trips.txt: Unrecognised column in trips.txt: vehicle_id
  • trips.txt: Unrecognised column in trips.txt: variant_id
  • variants.txt: Unrecognised file in feed: variants.txt
  • vehicle_types.txt: Unrecognised file in feed: vehicle_types.txt
